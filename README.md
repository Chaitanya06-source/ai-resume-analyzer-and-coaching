# 🚀 AI Resume Analyzer & Coaching System

A full-stack AI-powered resume analysis and career coaching application that helps job seekers optimize their resumes, identify career paths, and receive personalized coaching advice.

## ✨ Key Features

- **📄 Resume Upload**: Upload PDF/DOCX/TXT resumes
- **🔍 AI-Powered Analysis**: Get detailed resume scoring using OpenAI GPT-4
- **📊 Smart Scoring**: Overall score (0-100) + ATS compatibility score
- **✅ Strengths & Weaknesses**: AI identifies what's working and what's not
- **🎯 Career Suggestions**: Personalized career path recommendations with match percentages
- **📚 Skill Gap Analysis**: Identify missing skills and learn what you need
- **🎓 Learning Recommendations**: Curated resources (courses, books, tutorials)
- **💬 Career Coaching**: Interactive AI career coach for Q&A
- **📜 History Tracking**: View all past analyses and coaching sessions

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| Frontend | Streamlit (Python web framework) |

| AI/ML | OpenAI GPT-4o-mini, NLTK, scikit-learn |
